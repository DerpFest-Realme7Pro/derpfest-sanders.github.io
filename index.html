A Brief about the Project

AOSiP Derpfest is an slightly modified AOSiP (Android Open Source Illusion Project) having all the original
as well as handpicked features ensuring the smoothest and aesthetic experience in the terms of Quality and
Stability that a user expects. We are dedicated to bringing you the best Android has to offer.

Up to Date Source

We always keep our sources up-to-date with timely merged Security patches and Frequent Updates. We try to keep the source clean and efficient to provide the best User experience possible.

Simplified, yet efficacious

We try to keep the ROM as simple as possible with constructive changes and implementations that makes every build as fruitful as possible for the users, So that they can utilise the ROM to it's fullest.
