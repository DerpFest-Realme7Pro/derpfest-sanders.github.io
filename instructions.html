Instructions

Precautions

• Make sure that you backup Persist and EFS.

• If encrypted, make sure you decrypt your device.

Steps to Install

• Wipe Dalvik/ART cache, Cache, Data and System

• Flash the latest Derpfest build. (don't flash gapps, it's included)

• Boot & enjoy ❤️.

• Make sure you flash Magisk after the first boot if you want the Root Access.

Safety Net

• Safety net passes without magisk.

• If you flash magisk to root, just go to settings of magisk manager, and turn on magisk hide. (this needs to done as per new magisk/google policies for passing safetynet on magisk)

Note

• It's an Alpha build, some customizations under DerpQuest might not work.
