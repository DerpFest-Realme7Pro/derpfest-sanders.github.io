Flashing Instructions

Precautions

• Make sure that you backup Persist and EFS.

• If encrypted, make sure you decrypt your device

Steps to Install

• Wipe Dalvik/ART cache, Cache, Data and System

• Install the latest AOSiP Derpfest build.

• Wipe cache/Dalvik and Reboot. If you get any pixel setup fc, Flash DerpFest 9.0 and them dirty flash 10.0 over it
